# 厝邊快報｜里長專用・一句話生成公告圖文

正修科技大學招生處 × USR 敬贈里辦公處使用的公告產生器。
選類別 → 說重點 → 按「產生公告」，即可取得**正式公告文字、LINE 版文字、1080×1350 公告圖卡**。

## 📁 檔案清單

| 檔案 | 說明 |
|---|---|
| `index.html` | 完整網頁（單一檔案，含全部程式與樣式） |
| `README.md` | 本說明文件 |

## 🚀 上傳到 GitHub Pages（三步驟）

1. 在 GitHub 開啟（或使用既有的）repository，點 **Add file → Upload files**，把 `index.html` 與 `README.md` 拖進去，按 **Commit changes**。
2. 進入 repo 的 **Settings → Pages**，Source 選 **Deploy from a branch**，Branch 選 `main`（資料夾 `/root`），儲存。
3. 約 1 分鐘後，網頁就會出現在 `https://<帳號>.github.io/<repo名稱>/`。之後只要重新上傳 `index.html` 覆蓋，即完成更新。

## 🎨 「AI 插圖」怎麼運作（免開 ChatGPT 網頁）

按下「🎨 AI 插圖」後，系統會**以「公告內容＋里長想補充的話」為畫面主題**，自動生成插圖並直接插入公告圖卡，右下角以小字標明來源。共有兩種引擎：

### 模式一：內建免費引擎（預設，零設定）
- 不需要任何金鑰、不需要註冊，上傳 GitHub 後直接可用。
- 來源失敗會自動遞補備用來源；若網路環境全部擋掉，會提示改用「📷 放照片」。

### 模式二：ChatGPT（DALL·E 3）生成（選用，品質較佳）
1. 到 <https://platform.openai.com/api-keys> 建立一組 OpenAI API 金鑰（`sk-` 開頭）。此為**付費服務**，DALL·E 3 每張橫式圖約 US$0.08。
2. 開啟網頁右下角「⚙️ 里的資料」，把金鑰貼入「OpenAI API 金鑰」欄位並儲存。
3. 之後按「🎨 AI 插圖」就會直連 ChatGPT 家的 DALL·E 3 生成，完成後自動插入圖卡，小字標明「插圖由 ChatGPT 生成」。
4. 若金鑰失效、額度不足或逾時，系統會**自動退回免費引擎**，不會卡住。

### 🔐 金鑰安全（重要）
- 金鑰**只儲存在該台裝置的瀏覽器（localStorage）**，不會上傳到 GitHub、也不會傳給任何第三方；每台要使用 ChatGPT 模式的裝置需各自填一次。
- **絕對不要**把金鑰寫進 `index.html` 或任何會 commit 到 GitHub 的檔案——公開 repo 中的金鑰會立刻被盜用。
- 建議在 OpenAI 後台為此金鑰設定每月用量上限（例如 US$5）。

## 🖼️ 其他功能備忘

- **📷 放照片**：里長自行上傳照片，維持原始長寬比完整置入，不裁切、不變形，也不會加 AI 署名。
- **🔀 換個版面**：五種版面（經典置中／色帶大標／直書紅幅／雜誌左對齊／斜角色塊）輪換，全部經過防重疊排版檢查。
- **⚙️ 里的資料**：里名、里長姓名、電話、LINE 等填一次，之後每張圖卡自動帶入落款；可切換是否顯示「正修招生處提供」署名。
- 公告文字（含 AI 撰寫）一律以里長輸入的「公告內容＋補充」為主體，不會自行加入無關的祝福語或提醒。
